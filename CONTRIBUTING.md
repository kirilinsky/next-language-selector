# Contributing ⌑

Thanks for helping out. This lib exists to save time, so let's keep the process efficient.

### 🛠 Quick Start

1. **Clone & Install**: We use `pnpm`.

   ```bash
   pnpm install
   ```

2. **Develop**

   ```
   pnpm dev
   ```

## 📜 Rules

- Zero Dependencies: This is a hard requirement. No new packages.

- Testing: Ensure your changes are stable.

- AI Code: If you used an LLM, verify the logic.

- Code Style: Run pnpm lint before pushing.

## 🚀 PR Process

```
git checkout -b feat/your-feature

git commit -m 'feat: description'

Open a PR and explain the "why".
```
